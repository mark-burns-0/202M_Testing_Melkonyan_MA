ЧЕК-ЛИСТ ДЛЯ ТЕСТИРОВАНИЯ ФУНЦИИ РЕШЕНИЯ КВАДРАТНОГО УРАВНЕНИЯ ax² + bx + c = 0
================================================================================

1. Два различных корня (дискриминант > 0)
   Описание: Проверка базового случая с двумя рациональными корнями
   Вход: a=1, b=-3, c=2
   Ожидаемый результат: (1.0, 2.0)

2. Один корень (дискриминант = 0)
   Описание: Проверка случая, когда корень единственный (кратности 2)
   Вход: a=1, b=-2, c=1
   Ожидаемый результат: (1.0, 1.0)

3. Нет действительных корней (дискриминант < 0)
   Описание: Проверка обработки отрицательного дискриминанта
   Вход: a=1, b=0, c=1
   Ожидаемый результат: Корней нет (или выброс исключения)

4. Линейное уравнение (a = 0, b ≠ 0)
   Описание: Проверка вырожденного случая, когда уравнение становится линейным
   Вход: a=0, b=2, c=-4
   Ожидаемый результат: (2.0, 2.0)

5. Линейное уравнение с одним корнем (a = 0, b ≠ 0)
   Описание: Другой пример линейного уравнения
   Вход: a=0, b=3, c=6
   Ожидаемый результат: (-2.0, -2.0)+

6. Вырожденный случай - нет переменной x (a = 0, b = 0, c ≠ 0)
   Описание: Уравнение превращается в неверное равенство (c = 0)
   Вход: a=0, b=0, c=5
   Ожидаемый результат: Корней нет

7. Вырожденный случай - тождество (a = 0, b = 0, c = 0)
   Описание: Уравнение превращается в тождество 0 = 0 (бесконечно много решений)
   Вход: a=0, b=0, c=0
   Ожидаемый результат: Бесконечно много решений

8. Иррациональные корни
   Описание: Проверка работы с корнями, содержащими иррациональные числа
   Вход: a=1, b=0, c=-2
   Ожидаемый результат: (-1.4142135623730951, 1.4142135623730951)

9. Дробные коэффициенты
   Описание: Проверка работы с дробными числами
   Вход: a=0.5, b=-1.5, c=1
   Ожидаемый результат: (1.0, 2.0)

10. Отрицательный коэффициент a
    Описание: Проверка случая с отрицательным старшим коэффициентом
    Вход: a=-1, b=5, c=-6
    Ожидаемый результат: (2.0, 3.0)
